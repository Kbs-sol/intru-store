## ğŸ”§ How Everything Works Now

### Page Loading Flow

1. **User visits page** (e.g., `/`)
2. **DOMContentLoaded fires**
3. **initKonamiCode()** starts listening for key sequence
4. **checkAuth()** verifies if user is logged in
5. **Path detection** determines which page to render:
   - `/` â†’ `renderHomePage()`
   - `/shipping` â†’ `renderPolicyPage('shipping')`
   - `/returns` â†’ `renderPolicyPage('returns')`
   - `/faq` â†’ `renderPolicyPage('faq')`
   - `/terms` â†’ `renderPolicyPage('terms')`
   - `/privacy` â†’ `renderPolicyPage('privacy')`

### Homepage Rendering Flow

1. **renderHomePage() called**
2. **loadProducts()** fetches from `/api/products`
3. **productsCache** populated with 6 products
4. **HTML generated** with:
   - Hero section with animations
   - Our Story with stats
   - Products grid with all 6 products
   - Values section
5. **innerHTML updated** - content appears immediately
6. **Event handlers attached** - Add to Cart, Buy Now work

### Policy Pages Rendering Flow

1. **renderPolicyPage(slug) called**
2. **Axios GET request** to `/api/pages/{slug}`
3. **Response contains HTML content** from database
4. **Content injected** into page container
5. **Page displays** with proper styling
